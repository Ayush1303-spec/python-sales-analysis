# Sales Data Analysis using Python

This project analyzes a retail sales dataset using Python and the Pandas library. 
The goal is to explore sales patterns, identify top-performing categories and regions, 
and generate insights from the data.

## Tools Used
- Python
- Pandas
- Matplotlib
- Jupyter Notebook / VS Code

## Dataset
The dataset contains retail transaction data including:

- Order Date
- Product Category
- Product Name
- Region
- Sales
- Profit

The dataset contains over 9,000 sales records.

## Analysis Performed

The following analysis was performed using Python:

- Sales by Product Category
- Sales by Region
- Top 10 Products by Revenue

## Key Insights

- Technology category generates the highest total sales.
- Certain regions contribute significantly more to overall revenue.
- A small number of products drive a large portion of total sales.

## Visualization

Charts were created using Matplotlib to visualize sales trends and product performance.

## Files in this Project
